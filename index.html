Generador de Sonidos Avanzado con Web Audio API

Esta aplicación utiliza la Web Audio API para generar sonidos personalizados y visualizarlos en tiempo real.

Puedes ajustar varios parámetros como el tipo de oscilador, frecuencia, duración, volumen, reverberación, distorsión y envolvente ADSR.

El gráfico muestra la forma de onda del sonido actual en tiempo real.
